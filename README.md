# Monster Slayer

A simple battle game built while practicing **Vue basics**.

## Overview

**Monster Slayer** is a small browser game where the player fights a monster in turn-based combat.

The player can:

- Attack the monster
- Use a special attack
- Heal
- Restart the game

This project is mainly focused on practicing Vue concepts like:

- Reactive data
- Computed properties
- Watchers
- Methods
- UI updates based on game state

## Tech Stack

- HTML
- CSS
- Vue.js (CDN)
- JavaScript

## How to Run

Clone the repository

Open index.html in your browser
